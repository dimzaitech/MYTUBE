# **App Name**: MyTUBE

## Core Features:

- Responsive Header: A header component that adjusts its layout and elements (menu icon, logo, search, profile) based on the screen size.
- Collapsible Sidebar: A sidebar component with navigation links (Home, Trending, Subscriptions, etc.) that can be collapsed on smaller screens.
- Dynamic Grid: A video grid component that changes the number of columns (4, 3, 2) based on the device type (desktop, tablet, mobile).
- Video Cards: Display video previews (thumbnails, title, channel info, views, upload time, and duration badge) adapting to different screen sizes.
- Adaptive UI: Adjusts user interface based on device automatically.

## Style Guidelines:

- Primary color: YouTube Red (#FF0000) for emphasis and branding.
- Background color: Black (#0F0F0F) to create a modern dark theme.
- Text color: White (#FFFFFF) for clear visibility against the dark background.
- Card background: Dark gray (#272727) for content separation.
- Body and headline font: 'Inter' (sans-serif) for clean readability across devices.
- Utilize a CSS Grid for the main content area to ensure responsiveness and easy column adjustments.
- Implement subtle animations for sidebar transitions and hover effects on video cards.
